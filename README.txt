This project is for CS2132.
1. In order to run this, please install NodeJS from: https://nodejs.org/
2. Change the user and password in dbOperations.js for the postgres db, in the form :"postgres://username:password@localhost:5432/DatabaseName";
3. Open command line and go to this directory
4. run "node server.js" in command line without quotations
5. go to localhost:3001 to see the website
